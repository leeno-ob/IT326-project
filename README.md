# IT326 Data Mining Project

## Project Description:
The project uses data mining techniques to analyze the students academic performances based on their exam scores, understanding the students exam scores is important because it helps identify these student who may need some additional academic support and to analyze those who are performing well. The goal is to classify these students based  on their performances and group students with similar exam results using clustering.

## Project Motivation:
The motivation for this project is that analyzing exam scores is important to understand students knowledge. Using classification techniques helps support better educational decisions and improve teaching strategies.
